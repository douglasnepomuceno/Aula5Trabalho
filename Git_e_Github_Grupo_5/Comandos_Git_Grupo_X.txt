git init
Este comando é usado para criar um novo repositório GIT. Local

git add
O comando git add pode ser usado para adicionar arquivos ao índice do git local 

git clone
O comando git clone clona um repositorio para maquina local 

git commit
O comando git commit é usado para confirmar as alterações feitas nos arquivos


git status
O comando git status exibe a lista de arquivos alterados juntamente com os arquivos que ainda não foram adicionados a camada de stage

git push Envia as alterações feitas para o ramo/brach repositório remoto associado ao diretório de trabalho

Exemplo
git push origin master

git checkout
O comando git checkout pode ser usado para criar ramos/branch
